ic-poll - The Remove After Attribute

Summary

The ic-remove-after attribute tells Intercooler to remove the element it is on after a given interval.

The removal will by done by the ic-transition that applies to the given element.

Syntax

The value of the attribute should a valid integer, followed by the string "s" (for seconds) or "ms" (for milliseconds). (e.g. ic-poll="100ms" indicates that the element should be removed after 100 milliseconds.)

Dependencies

ic-remove-after has no dependency implications.
